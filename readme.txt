== Description ==

<p>Are you worried about the spam? Is Akismet too strict? Yeah, SidXSpam is easy and simple!Try it now!</a>
<p>SidXSpam is an anti-spam plugin for Wordpress.The plugin works when reviewers submit replys,and then check the Verify Code.If the code does not correct it cannot comment.</p>

<br /><br /><br />
<p>讨厌垃圾评论？Akismet误判您的正常评论? 试试简单有效的SidXSpam吧！</p>
<p>SidXSpam是一个防垃圾评论的Wordpress插件。该插件在用户输入评论时需要输入随机验证码，提交后进行验证。如果验证码不正确则不能发表评论。</p>

== Installation ==

<li>Upload the folder `/sidxspam/` to the `/wp-content/plugins/` directory</li>
<li>Activate the plugin through the 'Plugins' menu in WordPress</li>
